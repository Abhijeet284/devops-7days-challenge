# 🚀 DevOps 7-Day Challenge

This repository documents my **7-day journey into DevOps**, where I build and deploy applications step-by-step while learning industry-relevant tools and practices.

## 🎯 Objective
To understand and implement the fundamentals of DevOps by creating a **complete CI/CD pipeline from scratch**.

## 🛠️ Tools & Technologies
- AWS (EC2)
- Linux & Bash
- Git & GitHub
- SSH
- Apache Maven & Java
- VS Code (Remote SSH)
- Docker (upcoming)
- CI/CD tools (upcoming)

## 📅 Challenge Breakdown

- **Day 1:** EC2 Setup & Web App Deployment  
- **Day 2:** Version Control with Git & GitHub  
- **Day 3:** Linux & Shell Scripting Basics  
- **Day 4:** Build Automation (Maven)  
- **Day 5:** Docker & Containerization  
- **Day 6:** CI/CD Pipeline Setup  
- **Day 7:** Final Deployment & Optimization  

## 📈 Outcome
By the end of this challenge, I aim to have a strong foundation in DevOps practices and a working CI/CD pipeline.

## 🚀 Motivation
This challenge is part of my journey to become a DevOps Engineer by gaining hands-on experience with real-world tools and workflows.
